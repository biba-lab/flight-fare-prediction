# ✈️ Flight Fare Prediction – Progetto di Analisi Dati

## 📘 Introduzione al progetto
Questo progetto analizza i **prezzi dei voli in India** con l’obiettivo di capire quali fattori influenzano il costo di un biglietto e costruire un modello in grado di **prevedere il prezzo** di un volo.

Il dataset contiene oltre **10.000 voli**, ciascuno con informazioni su:
- compagnia aerea  
- città di partenza e arrivo  
- numero di scali  
- durata del volo  
- orari di partenza e arrivo  
- data del viaggio  
- prezzo finale  

È un problema reale, molto comune nel settore dei trasporti e delle piattaforme di prenotazione voli.

---

## 🎯 Obiettivo del progetto
L’obiettivo principale è:

👉 **prevedere il prezzo di un volo** a partire dalle sue caratteristiche.

Il progetto include:
- Data Cleaning  
- Feature Engineering  
- Analisi Esplorativa (EDA)  
- Modellazione con Machine Learning
